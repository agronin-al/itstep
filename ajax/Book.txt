Структура UNIX систем (папки):

etc - хранится конфигурация системы
home - учетные записи системы
usr - хранятся пользовательские файлы
root - файлы администратора
bin - хранятся все исполняемые файлы (системные программы)
var - хранит данные
    ./log- хранит логи
    ./temp
    ./www
local - локальные данные
opt - хранятся ваши программы
dev - сокеты для всех девайсов подключенных к компьютеру


Команды для консоли:

ls - выводит список фалов и папок в данной дириктории
cd - переход по папкам
mc - открывает текстовое окно
nano root/text.txt -позволяет редактировать файл
mcedit - запускает редактор для манипулсяции с файлами
mkdir - создание папки
rm - удаление файла
cp - копирование файла
ln from 1 to - создание ссылки
reboot - перезагрузка сервера
halt - выключение
kill - убить программу
whereis - поиск
df - h  - сколько свободного места в дирикториях
useradd
password

chmod - устанавливает права
apt-get - установить программу
etc/apt/source.list
apt-get update - обновить базу данных программ программу
apt instal установить программу
apt purge - удаляет программу вместе с конфигурациями
apt remove - удаляет только программу
